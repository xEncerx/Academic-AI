# Правила текстового оформления по ГОСТу

## Обзор

Этот навык задаёт правила именно для текстового оформления академической работы по ГОСТу и по методическим указаниям вуза. Он применяется при написании нового текста, редактировании существующего текста и финальной проверке перед переносом в документ.

## Когда использовать

Используй навык, если нужно:

- написать фрагмент академической работы сразу по правилам текстового оформления;
- отредактировать уже готовый текст под ГОСТ;
- проверить черновик на нарушения пунктуации, кавычек, сокращений, ссылок, списков, формул и библиографических записей;
- устранить конфликт между «естественным» стилем текста и формальными требованиями методички.

Не используй этот навык для Typst-макросов, структуры шаблона, нумерации элементов документного шаблона или технической вёрстки. Этот навык отвечает за языковые и текстовые нормы, а не за синтаксис Typst.

## Порядок работы

1. Сначала определи тип фрагмента: основной текст, список, подпись, формула с пояснениями, ссылка на объект, библиографическая запись.
2. Затем проверь специальные правила для этого типа фрагмента.
3. После этого проверь общие правила: пробелы, кавычки, сокращения, дефис, тире, математические знаки.

## Общие правила текста

### Пробелы и знаки препинания

- Ставь пробел после любого знака препинания, но не перед ним.
- Не ставь пробел после открывающей скобки.
- Не ставь пробел перед закрывающей скобкой.
- Не ставь пробел после открывающей кавычки.
- Не ставь пробел перед закрывающей кавычкой.

Правильно:

```text
Анализ показал, что метод устойчив.
Результат представлен в таблице (Таблица 2.1).
Термин «модель данных» используется далее.
```

Неправильно:

```text
Анализ показал , что метод устойчив.
Результат представлен в таблице ( Таблица 2.1 ) .
Термин « модель данных » используется далее.
```

### Кавычки

- В русском тексте используй только кавычки-ёлочки: «…».
- В английском тексте используй только двойные кавычки: "…".
- Не смешивай разные типы кавычек внутри одного русскоязычного фрагмента без необходимости.

### Сокращения

- Не сокращай слова произвольно.
- Разрешены только общепринятые сокращения.
- Разрешены сокращения, для которых ранее в тексте дана полная расшифровка.
- При сомнении выбирай полную форму.

### Дефис, минус, тире

- Дефис «-» используй без пробелов внутри сложных слов и устойчивых форм: физико-математический, во-первых, кто-либо.
- Знак минуса «–» используй как знак вычитания.
- Длинное тире «—» используй как знак пунктуации; оно отделяется пробелами с двух сторон.
- Не удаляй длинное тире только ради «очеловечивания» текста.
- Заменяй длинное тире только тогда, когда оно реально нарушает смысл, пунктуацию или норму оформления.

Правильно:

```text
Экономика — это хозяйственная деятельность общества.
Во-первых, необходимо определить критерии оценки.
```

Неправильно:

```text
Экономика—это хозяйственная деятельность общества.
Во - первых, необходимо определить критерии оценки.
```

### Математические знаки в обычном тексте

- Знаки «+», «–», «=», «>», «<» и аналогичные используй только в формулах.
- В обычном тексте пиши их словами: «плюс», «минус», «равно», «больше», «меньше».

Правильно:

```text
Темп роста оказался больше прогнозного значения.
```

Неправильно:

```text
Темп роста оказался > прогнозного значения.
```

## Ссылки в тексте

### На рисунки, таблицы, листинги, приложения

- Ссылки оформляй в круглых скобках или встраивай в предложение.
- Пиши слова «Рисунок», «Таблица», «Листинг», «Приложение» с большой буквы.
- Всегда указывай порядковый номер объекта.

Примеры:

```text
Результаты расчёта приведены в Таблице 5.4.
Схема показана на Рисунке 2.
Подробности представлены в Приложении А.
(Таблица 5.4).
```

### На рисунки при разных схемах нумерации

- При сквозной нумерации: «… в соответствии с Рисунком 2».
- При нумерации в пределах раздела: «… в соответствии с Рисунком 1.2».

### На формулы

- Ссылку на номер формулы давай только в круглых скобках.
- Используй форму: «… в формуле (7.1)».

### На источники

- Ссылки на использованные источники давай арабскими цифрами в квадратных скобках.
- Используй формы: [1.5], [2.18], [2.18, c.21-25].
- После ссылки в конце предложения ставь точку.

Правильно:

```text
Метод уже применялся в близких задачах [2.18].
Подход подробно описан ранее [2.18, c.21-25].
```

## Списки

### Маркированные списки

- Каждый пункт начинай со строчной буквы.
- Каждый пункт заканчивай точкой с запятой.
- Последний пункт заканчивай точкой.

Пример:

```text
- первый критерий оценки;
- второй критерий оценки;
- третий критерий оценки.
```

### Нумерованные списки

- Каждый пункт начинай с прописной буквы.
- Каждый пункт заканчивай точкой.

Пример:

```text
1. Первый этап анализа.
2. Второй этап анализа.
3. Третий этап анализа.
```

## Формулы

### Пунктуация после формулы

- Если формула завершает предложение, после неё ставь точку.
- Если после формулы предложение продолжается, после неё ставь запятую.

### Перенос формулы

- Если формула не помещается в одну строку, переноси её после знака равенства (=) или после знаков плюс (+), минус (-), умножения (×) или деления (:).
- Знак, после которого выполнен перенос, повторяй в начале следующей строки.

### Пояснения под формулой

- Пояснения располагай непосредственно под формулой.
- Сохраняй ту же последовательность символов и коэффициентов, что и в формуле.
- Первую строку начинай словом «где» без двоеточия.
- Слово «где» пиши без абзацного отступа.
- Каждый символ и коэффициент поясняй с новой строки.
- Символы располагай один под другим.

Правильно:

```text
где T0 — трудовые затраты по базовому варианту;
    T1 — трудовые затраты по предлагаемому варианту.
```

## Таблицы

- Не включай в таблицу графу «№ п/п».
- Если нужна нумерация показателей или других данных, указывай порядковые номера в левом столбце перед наименованием показателя.

## Листинги

- При ссылке на отдельные строки программы используй нумерацию строк по возрастанию с шагом 10.
- После ссылки на листинг в фигурных скобках указывай номера строк.
- Допустимы формы: «Листинг 3.2 {30, 50, 140}», «Листинг 3.2 {20-80}».

## Список использованных источников

- Список использованных источников составляй по разделам.
- Внутри каждого раздела располагай источники в порядке их упоминания в пояснительной записке.
- В библиографических описаниях учитывай специальные сокращения городов: Москва — «М.», Ленинград — «Л.», Санкт-Петербург — «СПб.», Киев — «К.».

### Исключения по пробелам в библиографии

Не применяй общее правило о пробеле после знака препинания механически в следующих случаях:

- сокращения вида «М.:», «СПб.:» в списке использованных источников;
- запятая после точки в инициалах автора.

Эти формы допустимы и не должны автоматически исправляться как ошибочные.

## Частые ошибки

- Замена «ёлочек» на обычные кавычки в русском тексте.
- Использование математических знаков в обычном тексте вместо слов.
- Постановка пробела перед запятой, точкой, двоеточием или закрывающей скобкой.
- Отсутствие пробелов вокруг длинного тире.
- Написание ссылок в виде «рис. 2», «табл. 3» вместо полной формы с заглавной буквы.
- Использование произвольных сокращений без расшифровки.
- Оформление маркированного списка с заглавных букв и точками в каждом пункте.
- Добавление в таблицу графы «№ п/п».
- Неправильное оформление ссылок на формулы и источники.
- Автоматическое исправление «М.:» или «И.И.,» как будто это ошибка пробелов.

## Чек-лист проверки

- [ ] После знаков препинания стоят пробелы, перед ними пробелов нет.
- [ ] После открывающих скобок и кавычек нет пробелов; перед закрывающими тоже нет.
- [ ] В русском тексте используются «ёлочки».
- [ ] Нет произвольных сокращений.
- [ ] Дефис, минус и тире различаются по функции.
- [ ] Математические знаки вне формул записаны словами.
- [ ] Ссылки на рисунки, таблицы, листинги, приложения оформлены с заглавной буквы и номером.
- [ ] Ссылки на формулы даны в круглых скобках.
- [ ] Ссылки на источники даны в квадратных скобках.
- [ ] После ссылки в конце предложения стоит точка.
- [ ] Маркированные списки начинаются со строчной буквы и оформлены через точку с запятой.
- [ ] Нумерованные списки начинаются с прописной буквы и оканчиваются точкой.
- [ ] Пояснения под формулой начинаются словом «где» без двоеточия.
- [ ] В таблицах нет графы «№ п/п».
- [ ] Ссылки на строки листинга оформлены в фигурных скобках.
- [ ] В библиографии сохранены исключения «М.:», «СПб.:» и запись инициалов.

## Антипаттерны

- Не запрещать длинное тире как класс знаков.
- Не заменять все тире запятыми или точками автоматически.
- Не сокращать «Рисунок», «Таблица», «Листинг», «Приложение» в ссылках.
- Не использовать математические символы в обычных фразах.
- Не унифицировать библиографию по общим правилам текста без учёта исключений.
- Не применять одно и то же правило ко всем фрагментам без учёта их типа.
