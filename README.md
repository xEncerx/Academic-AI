# 📝🎓 Academic-AI: Typst шаблон для написания отчетов/дипломов/курсовых работ(ИИ совместимый)

Готовый шаблон, который оформлен по требованиям [РТУ МИРЭА](./assets/Методические%20указания%20по%20оформлению%20работ.pdf)

## 📦 Установка Typst

Для работы с шаблоном вам понадобится [Typst](https://github.com/typst/typst).

<details>
<summary>🪟 Windows</summary>

### Установка на Windows

Выполните команду в PowerShell или CMD:

```powershell
winget install --id Typst.Typst
```

</details>

<details>
<summary>🐧 Linux</summary>

### Установка на Linux

Выполните команду в терминале:

```bash
curl -fsSL https://typst.community/typst-install/install.sh | sh
```

</details>

<details>
<summary>🍎 macOS</summary>

### Установка на macOS

Выполните команду в терминале:

```bash
brew install typst
```

</details>

## ✍️ Написание работы

Работу можно написать двумя способами: **вручную** или с помощью **ИИ-ассистента**.

### Вариант 1: Вручную

Если вы предпочитаете писать самостоятельно:

1. **Изучите документацию:**
   - [Документация по шаблону ГОСТ](./docs.md) — подробная инструкция по использованию макросов, вставке таблиц, рисунков и формул.
   - [Официальная документация Typst](https://typst.app/docs/reference) — полный справочник по синтаксису.

2. **Отредактируйте необходимые файлы:**
   - `chapters/02_main_text.typ` — основной текст работы
   - `chapters/03_references.typ` — список литературы
   - `chapters/04_appendix.typ` — приложения (если нужны)
   - Добавьте изображения в `assets/images/`
   - Добавьте код в `assets/codes/`

3. **Замените титульный лист:**
   - Если у вас есть готовый титульный лист — замените файл `assets/title_page.pdf`
   - Если титульный лист не нужен — просто удалите файл

### Вариант 2: С помощью ИИ

Агент автоматически подгрузит инструкции из папки `.agents/skills/` и будет знать, как оформлять работу по ГОСТу. Просто опишите, что хотите добавить, и агент сам применит правильное форматирование.

**Доступные скиллы:**

- `format-typst-gost` — форматирование по ГОСТу (таблицы, формулы, рисунки, списки литературы и т.д.)
- `write-human-academic-text` — написание "человеческого" академического текста
- `gost-text-rules` — проверка и соблюдение правил ГОСТ при написании текста (пунктуация, кавычки, сокращения, оформление списков и ссылок)
- `humanize-academic-text` — "очеловечивание" уже написанного текста

## 🚀 Быстрый старт

1. **Установите Typst** (см. раздел [Установка](#-установка-typst))

2. (Опционально) Для удобной работы с проектом, установите `Vs code` и расширение `Tinymist Typst`

3. **Склонируйте/скопируйте шаблон:**

   ```bash
   git clone https://github.com/xEncerx/Academic-AI
   cd Academic-AI
   ```

4. **Начните писать!**
   - Вручную — редактируйте файлы в `chapters/`
   - Или вызовите ИИ-агента и описывайте, что нужно добавить

5. **Скомпилируйте PDF:**
   ```bash
   typst compile main.typ main.pdf
   ```
