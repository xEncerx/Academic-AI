# GOST Template — ИИ-агент координатор

## Обзор проекта

ИИ-совместимый шаблон для написания курсовых, дипломов и отчетов по ГОСТу в Typst. Проект содержит готовые макросы и скиллы для автоматического оформления академических работ.

## Карта скиллов

| Запрос пользователя | Какой скилл вызвать |
|---------------------|---------------------|
| Написать текст с нуля (курсовая, реферат, глава) | `write-human-academic-text` + `format-typst-gost` + `gost-text-rules` |
| Отформатировать .typ файл, вставить таблицу/рисунок/формулу | `format-typst-gost` |
| Проверить текст на соответствие правилам ГОСТ (пунктуация, кавычки, сокращения) | `gost-text-rules` |
| «Очеловечить» уже написанный ИИ-текст | `humanize-academic-text` |

**Важно:** При написании текста с нуля (использование `write-human-academic-text`) необходимо также загрузить скиллы `format-typst-gost` (для правильного форматирования в Typst) и `gost-text-rules` (для соблюдения правил пунктуации и оформления текста). Скиллы дополняют друг друга и должны применяться комплексно.
